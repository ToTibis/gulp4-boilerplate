In this boilerplate you can use:

  • - HTML - processing using Pug.js;
  • - Sass - Bootstrap 5 and some custom things
  • - Javascript ES6 Modules - processing using Webpack 5;
  • - Without jQuery - instead in this template included set of helpers-functions written on vanilla javascript ($data, $dom, $events, $style, $ui) and Bootstrap 5 components - Offcanvas, Collapse, Dropdown, Modal etc;
  • - SVG-sprites.
  • - Ready-made project structure
  • - Images lazy load by - vanilla-lazyload
  • - Form validation by - validator
  • - Modal - by Bootstrap5 with controller wrap - you can call modal window programmatically and create modal with dynamic content
  • - Toasts - based on toastify
